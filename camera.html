PTZ arrows: 30×30px inside preset-style button tiles. Zoom/tracking: 30×30px. Left-panel camera and tracking buttons: 30×30px icons above 16px label text, matching Room Preset button sizing.

Universal left-panel button style — 16px / weight 700, 60px min-height

All left-panel buttons across Camera, Lighting, and Audio pages use the same consistent style as the Display page's display-btn, scaled up to 16px / weight 700 for improved readability:

  • Layout: Flex row — 16px icon inline-left inside the label, text left-aligned, right side free.
  • Size: Full width, padding: 10px 14px, border-radius: 10px, min-height: 60px.
  • Default: #DDE1E7 background, muted text.
  • Active/selected: #2D49E8 blue fill, white text + icon — consistent with display.html.
  • Chevron: Camera selector buttons include a right-facing chevron (opacity 0.4 default, full on active). Tracking buttons have NO chevron — they are binary toggles, not navigation.
  • Sub-label: Camera selector shows a secondary line ("Active camera" / "Tap to switch") at 11px. Tracking buttons show label only.

This button style is defined system-wide. Do not deviate from these values on any left-panel button across any page.

4

PTZ Control Buttons

Directional pad — pan and tilt controls

3×3 grid, four cardinal arrow buttons and a blue centre button. Each button: minimum 60–70px square, 12px radius. Press-and-hold to move, release to stop.

  • Arrow buttons: #DDE1E7 background, dark chevron icon
  • Centre/Home: #2D49E8 blue, white dot indicator
  • Zoom In/Out buttons sit in a separate column to the right

Saved position presets — horizontal row, fills blue when active

Presets are a horizontal flex row of equal-width buttons. Camera icon (24px) above preset name (17px / weight 700). Active preset fills #2D49E8 blue with white icon and text — consistent with the camera and tracking selection buttons.

  • Minimum 64px tall, 12px border radius
  • Default: #DDE1E7 background, dark text/icon
  • Active: #2D49E8 blue, white text/icon, box-shadow
