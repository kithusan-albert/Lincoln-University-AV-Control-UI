Microphone Control Page — Design Principles

Lincoln University AV Control — Audio (Microphone) Control — interface specification

Rule of Thumb: Global UI design (Colour Palette, Header Bar, Footer Navigation Bar, Typography, Shutdown Modal) is documented on the Display page. This page covers Audio-specific elements only.

Stroke-only SVG icons — Feather/Lucide, fill="none", stroke="currentColor"

Master Control button: 30×30px icon. Channel row mute icons: 20×20px. Footer: 24×24px, sw 1.8. Power: 30×30, sw 2.2. Muted state: stroke colour switches to #ff6b6b.

  • Mic Active — capsule + arc + stand. Master Control button, channel row, footer Mic Mute.
  • Mic Muted — same icon + diagonal strike-through, sw 2. Master Control when all muted.
  • Vol Mute / Low / High — speaker cone variants. Footer vol-cluster.
  • Power — arc + vertical stem. Footer Shutdown, 30×30, sw 2.2.
2

Page Layout

Two-column layout — status & master control left, mic channels right

Left panel (340px): live mic status badges and the Master Control (Mute All) button. Right panel: stacked mic channel rows, one per microphone. The footer is identical to all operational pages — see Display page for specification.

3

Left Panel — Master Control Button

Mute All Mics
Mute All Mics

Universal left-panel button style — 16px / weight 700, 60px min-height, fills blue when active

The Master Control button uses the same universal left-panel style as all buttons across Camera, Lighting, and Audio pages: icon inline-left (16×16px), 16px / weight 700 label, min-height: 60px, border-radius: 10px. No chevron — this is an action button, not a navigation selector.

  • Default: #DDE1E7 background, muted icon and dark text — reads as ready to be tapped
  • All muted (active): #2D49E8 blue fill, white icon and text, box-shadow — same active fill as camera and display left-panel buttons, reinforcing system-wide consistency
  • Tapping also updates all individual channel mute states in sync

This button is wrapped in a white card panel with a "Master Control" label above it. The card styling (white background, 14px border radius, 1.5px #D1D8E0 border) matches group cards used on the Display and Camera pages.

Four-element horizontal row — icon, name, pill slider, mute button

Each microphone channel occupies a full-width row: icon square (44×44px), name label (15px / weight 700, min-width 90px), pill-style level slider, and a mute toggle button (52×52px). When muted, the row background shifts to #FEF2F2 with a red border, and the slider drops to 45% opacity and becomes non-interactive.
