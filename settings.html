3-column grid with a "Settings" section heading

The main content area uses a 3-column grid at 1fr 1fr 1fr with 14px gap. Cards use align-items: stretch so cards in the same row match height exactly.

A bold uppercase section heading — "Settings" — sits above the grid with 16px / weight 700 / letter-spacing 0.08em / text-transform uppercase, matching the "Camera Controls" heading style on the Camera page. It gives the page a clear label without a separate header strip.

  • 1-column card: Projector Mode occupies one column.
  • 2-column card: Room Combining spans two columns via .span-2.
  • Card style: White background, 14px border-radius, 1.5px #D1D8E0 border, 14px 16px 16px padding. No icon, no divider line, no description — just a small muted uppercase card title.

Two horizontal buttons filling full card height

Projector Mode uses two proj-btn buttons arranged in a horizontal row (display: flex; gap: 10px) that each flex: 1 to share the available width equally. The card itself is a flex column so the button row stretches to fill the full card height — making the Projector Mode card the same height as the Room Combining card beside it.

Each button shows a centred SVG icon above a two-line label ("Single / Mode", "Dual / Mode"), using flex-direction: column; align-items: center.

  • Default state: #DDE1E7 background, dark text, dark icon stroke
  • Active state: #2D49E8 blue fill, white text and icon, subtle box-shadow
  • Gap between buttons: 10px — same as the room status tile grid gap
  • No diagram, no preset sub-labels — the buttons are the only control

Status tiles + merge buttons — no diagram, no icons

Room Combining spans 2 columns. It contains two layers stacked vertically: a row of five status tiles (D1–D5) at the top, and a row of five "Merge" buttons below.

Status tiles are a 5-column grid with 10px gap. Each tile shows the room name (bold) and a single status word at the bottom. There are exactly three states — only one shows at a time, in priority order:

  • Combined — red tint: #FFF5F5 background, #E53E3E border, #C53030 text. Shown when the room has been merged with an adjacent room via the Merge buttons.
  • In Use — yellow tint: #FFFBEB background, #D97706 border, #B45309 text. Shown when the room has an active AV session (system-driven). Combined takes priority over In Use.
  • Standalone — neutral: white background, default border, muted text. Default state when not combined and not in use.

No pills, no badge overlays, no icons inside tiles.

Merge buttons are plain text-only buttons ("Merge D1" through "Merge D5"). Selected state fills blue. Selecting adjacent rooms automatically marks them Combined. No icons on the buttons.
