Lighting Control UI — Design Principles

Lincoln University AV Control — Lighting Control — interface specification

Rule of Thumb: Global UI design (Colour Palette, Header Bar, Footer Navigation Bar, Typography, Shutdown Modal) is documented on the Display page. This page covers Lighting-specific elements only.

1

Stroke-only SVG icons — Feather/Lucide, fill="none", stroke="currentColor"

Lighting control uses the bulb icon throughout. Quick preset and zone buttons all use 30×30px icons. Scene buttons in the left panel use 20×20px icons.

  • Lightbulb — standard bulb body + base line. Zone cards, footer Lighting nav button.
  • All Off — bulb with diagonal cross-through. Quick preset and All-off scene.
  • All On / rays — bulb + short ray lines (left, right, top-corners). Quick preset.
  • Teaching — open book icon. Left panel scene preset.
  • Event — monitor icon. Left panel scene preset.
  • Power — arc + vertical stem. Footer Shutdown, 30×30, sw 2.2.

Left Column — Room Scene Presets

Teaching
Event

Universal left-panel button style — 16px / weight 700, 60px min-height, blue when active

Scene buttons use the same universal left-panel style as all other left-panel buttons across Camera, Audio, and Lighting pages: icon inline-left (16×16px), text left-aligned (justify-content: flex-start), 16px / weight 700 label, min-height: 60px, border-radius: 10px. No chevron.

  • Default: #DDE1E7 background, muted text and icon
  • Active: Blue fill — #2D49E8 background, white icon, white text. Matches the universal blue active state used across all left-panel buttons in the system
  • Two scenes only: Teaching and Event. The "All lights off" scene was removed — use the Quick Preset "All Off" tile on the right panel for that function
  • Tapping a scene applies settings to all zones simultaneously. Active highlight clears if any zone is manually adjusted afterwards

Quick Presets — Right Panel

All Off
All On
Low

5-column preset grid — icon top, text below, blue when active

Quick Presets use the same preset-btn tile style as source buttons on the Display page: icon stacked above text, min-height: 108px, border-radius: 14px, 16px / weight 700. Five presets: All Off, All On, Low, Medium, High.

Active state uses blue (#2D49E8) — the same blue used by Scene Presets and Camera controls. This unifies the "selected" visual language for room modes.

  • Default: #DDE1E7 grey background, dark icon and text
  • Active: #2D49E8 blue fill, white icon and white text
  • The active highlight clears if any zone is manually adjusted afterwards
5

Switch Zones — Preset-Style Buttons

Downlights On
Stage Off

2-column grid — preset-button style, blue when On

Switch zones (circuits with no dimming) are shown as preset-style tiles: icon top, zone name, status label below. Same sizing as all other preset buttons (min 108px tall, 14px radius, 16px bold label).

  • Off: #DDE1E7 background, dark icon and text, status reads "Off"
  • On: #2D49E8 blue fill, white icon, white text, status reads "On"
  • Hover state: blue tint border, subtle blue background wash
  • Tapping anywhere on the tile toggles the zone On/Off

3-column grid — preset-button style, blue when On, slider below

Dimmer zones use the same tile shape as Switch Zones and Quick Presets, but include a brightness slider below the name/status text.

  • Off: #DDE1E7 background, dark icon/text, slider area at 35% opacity and non-interactive
  • On: #2D49E8 blue fill, white icon, white text, white slider (semi-transparent track, white thumb)
  • Tapping the upper portion (icon + name) toggles On/Off
  • Slider and −/+ step buttons control brightness percentage (shown in status label)
  • Step buttons: semi-transparent white rectangles when On; grey border when Off
