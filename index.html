Stroke-only SVG icons — Feather/Lucide, fill="none", stroke="currentColor"

All icons are inline SVG. No raster or filled icons are used. Footer nav icons: 24×24px, stroke-width="1.8". Power/Default footer buttons: 28–30px, stroke-width="2.2". Vol cluster: 24×24px, stroke-width="2.0".

  • Power — arc + vertical stem. Footer Shutdown, 30×30, sw 2.2, red background.
  • Welcome/Home — house silhouette with door rectangle. Footer and nav panel.
  • Display/Monitor — rectangle with stand. Footer Display button and nav.
  • Lighting — lightbulb body + base line. Footer Lighting button and nav.
  • Camera — rectangle body + right chevron. Footer Camera button and nav.
  • Audio/Mic — capsule + arc + stand. Footer Audio button and nav.
  • Settings/Gear — inner circle r=3 + 8-node outer path. Footer Settings and nav.
  • Default/Reset — CCW circular arrow. Footer Default, 28×28, sw 2.2, amber background.
  • Vol Mute / Vol Low / Vol High — speaker cone with X / one arc / two arcs. Footer vol-cluster, left to right.
  • Mic Muted — microphone with diagonal strike-through line (x1="3" y1="3" x2="21" y2="21"). Footer Mic Mute toggled state.

Canvas & Panel Size

1280 × 800 px — Landscape

1280 × 800 pixels, fixed landscape canvas

Same canvas size as all other pages — 1280 × 800 pixels, landscape, no scrolling. The outer page background matches all other pages at #1A1A2A. The panel itself uses #34333B — the same charcoal as the footer on other pages — so the welcome screen feels like a unified dark canvas rather than a white-panel interface.

Three vertical zones top to bottom:

  • Koru border — 52px. Darker charcoal background with a tiling kowhaiwhai spiral pattern.
  • Main content area — fills remaining height (~660px). Horizontally centred two-column layout.
  • Footer — 88px. Same #34333B as the panel — blends seamlessly.

Unified dark palette — no blue accent on this page

The welcome screen deliberately omits the blue header bar. The full panel is one dark tone, creating a calm idle state that contrasts with the active blue-accented operational pages.

  • Panel BG (#34333B) — panel body and footer share this colour exactly. The footer has no visual boundary, only a 1px line at rgba(255,255,255,0.07).
  • Koru Strip (#2A292F) — slightly darker, used only behind the top decorative band for subtle depth.
  • Room Name (#FFFFFF) — pure white, full opacity. The only fully white element on the screen — deliberate maximum contrast for the most important piece of information.
  • Greeting (75% white) — "Welcome to" is dimmer than the room name, reinforcing the hierarchy.
  • Mute Red (#FF6B6B) — identical to all other pages, used only when the mic is muted.

Typography — Welcome Text

Welcome to Room / Number
Press the screen to start

Three-level welcome hierarchy — light, bold, outlined

The welcome block uses three distinct sizes and weights to create rhythm. All three are horizontally centred as a group:

  • "Welcome to" — 32px / weight 300 (light). Colour: rgba(255,255,255,0.75). Letter-spacing: 0.02em. Intentionally smaller and lighter — this is the supporting label, not the primary read.
  • Room / Number — 46px / weight 700 (bold). Colour: #FFFFFF full opacity. The highest-contrast element on the page. Customise per room in Crestron via a text join or hardcoded string.
  • "Press the screen to start" — 17px / weight 500. Colour: rgba(255,255,255,0.78). Set inside a transparent pill with a 1.5px white border at 38% opacity that pulses gently between 38% and 68% on a 3-second loop to draw attention.

Gap between welcome text and CTA button: 32px.

The "Welcome to" line uses weight 300 specifically to contrast the bold room name below it. Do not increase this weight — the visual tension between the two lines is intentional and gives the greeting its elegance. The font size of 32px for "Welcome to" and 46px for the room name are the definitive values for this page.
5

Two-Column Layout

Horizontally centred two-column split with vertical divider

The main content area uses a horizontal flex layout centred in the full 1280px panel. The two halves are separated by a 1px vertical divider at 200px tall, rgba(255,255,255,0.14).

  • Left — Logo block: Vertically centred column with the placeholder crest (150×150px outlined box), then the wordmark below: "Lincoln" at 24px / weight 700 / ALL CAPS / letter-spacing 0.22em, "University" at 13px / weight 600, and "Te Whare Wānaka o Aoraki" at 11px / weight 400 / 32% opacity. Replace the placeholder box with the real university crest at the same bounding dimensions.
  • Right — Welcome block: All three elements (greeting, room name, CTA button) centred horizontally with 4px between the two text lines and 32px between the text block and the CTA button.

Gap between the two columns: 72px. Outer horizontal padding of the main area: 80px each side.

6

Footer — Blended, No Power Button

88px footer — same colour as panel, no power or nav buttons

The welcome screen footer differs from all operational pages in two ways:

  • No power block — the red shutdown button is absent. The system has not been started yet.
  • No navigation icons — no page nav buttons. The welcome screen is the entry point; page navigation only appears once the operator taps through to an active page.
  • Background matches panel — #34333B throughout, separated only by a 1px hairline at rgba(255,255,255,0.07). The footer visually dissolves into the panel.

What is present: Left group — light bulb icon and microphone icon at 35% white opacity (status indicators only). Right group — mic-mute toggle and three volume step buttons (mute, low, high), identical in behaviour to all other pages.

Both icon groups have a 1px vertical divider at rgba(255,255,255,0.08) on their outer edge. The left group borders right, the right group borders left. No centre divider is needed since there are no nav icons between them.
